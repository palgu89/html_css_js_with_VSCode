3. 문자데이터를 분석하여 단어 조합하여 새로 생성하기 특정 문자열 뒤
    
    혹은 앞에 문자를 추가하여 새로운 문자데이터를 만드는 방식
    
    마케팅이나 광고관련 프로그램에서 많이 사용하며 데이터 분석 분야에서
    
    데이터를 원하는 형식으로 전처리 할 때도 많이 사용됨
    
    substring, slice, replace 메서드 활용하면 좋아요~ 
    
    중첩반복문 활용하지 않아도 해결 가능
    
    
    strData의 개발자라는 단어 뒤에 '-백엔드'를 추가,
    
    관리자라는 단어 뒤에 '-고생하세요'를 추가
    
    위의 조건에 맞게 코드를 새로운 newData라는 변수에 저장해보기
    
    let strData = '자바개발자, 웹개발자, 앱개발자, 시스템관리자, 서버관리자';
    
    let newData = '자바개발자-백엔드, 웹개발자-백엔드, 앱개발자-백엔드,'
    
                   + '시스템관리자-고생하세요, 서버관리자-고생하세요';
