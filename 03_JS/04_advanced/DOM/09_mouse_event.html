자주 사용하는 마우스 이벤트

  • click: onclick, dbclick(더블클릭), contextmenu(우클릭)
  • wheel: wheel, scroll, wheel button click(?)
  • drag and drop: drag, [dragstart, draaagleave, dragover, dragenter, dragend,] drop
  • focus and blur: 폼태그의 커서 이동, 커서 아웃
  • input change: 입력된 상태, 입력한 값이 바뀌는 상태
  • mouseover(CSS hover): target element의 자식요소에까지 영향 미침
  • mouseenter(mouseleave): target element에만 이벤트 적용
  • submit: 폼태그가 제출될 때
