Position

  • fixed : 브라우저를 기준으로 특정 위치에 고정으로 배치
  • static (default) : 원래의 정상적인 흐름(위치)
  • relative : 정상적인 위치로부터 상대적인 위치에 태그(요소-element)를 배치
  • absolute : 부모태그가 relative일 때, 혹은 부모태그 중 relative가 없으면
    전체페이지(body)를 기준으로 시작위치에서 top, left, right, bottom 등의 절대적 수치만큼 배치
